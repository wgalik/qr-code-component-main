# Frontend Mentor - QR code component solution

This is my solution to the QR code component challenge on Frontend Mentor.
Frontend Mentor challenges help you improve your front-end skills by building real-world projects.

## Overview

[Live Site URL](https://wgalik.github.io/qr-code-component-main/)

## My process

Built with:

- Semantic HTML5 markup
- CSS custom properties
- CSS Grid
- Mobile-first workflow
- Google Fonts (Outfit)

## What I learned

This project helped me practice creating a simple and clean layout using CSS Grid and Flexbox.
I also improved my understanding of responsive design and centering elements vertically and horizontally using flexbox on the body element.
